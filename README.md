Baseee是一个跨平台c++程序库   
旨在扩展c++   
本项目分为以下模块：   
log 日志   
string  字符串处理   
欢迎捐款or贡献代码  

## Build
Baseee可以在很多平台构建，使用Cmake构建系统   
```
$ git clone https://github.com/chhdao/Baseee --depth=1
$ cd Baseee/baseee
$ mkdir Build
$ cd Build
$ cmake ..
$ cmake --build .
$ cmake --install .
```
即可完成源码安装。  
Enjoy!
